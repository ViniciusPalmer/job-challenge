# Job Challenge - Animal Search App

A React + TypeScript application built as part of a **2021 job interview challenge**. The project simulates an animal search system with fake in-memory data and context-based state management.

## Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: styled-components
- **Routing**: React Router v6
- **Testing**: Jest + React Testing Library
- **Linting**: ESLint

## Features

- Animal search with filters
- Fake in-memory data (fakerAnimals)
- Context-based state management
- Responsive layout
